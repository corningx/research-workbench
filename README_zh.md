# Research Workbench

[English](README.md) | 简体中文

一个面向研究生和科研人员的本地优先科研项目管理工作台。

无需安装、无需注册账号、无需服务器。下载后即可在浏览器中使用，研究数据默认保存在本地，不会主动上传。

## 主要功能

- 多研究项目管理
- 科学问题与研究假设记录
- 研究路线图
- 实验结果与异常记录
- 论文图件计划
- 相关文献管理
- 待办事项管理
- JSON 数据导入与导出
- 本地自动保存与恢复快照

## 使用方法

1. 下载仓库中的 `index.html`。
2. 使用 Chrome 或 Edge 浏览器打开。
3. 新建研究项目，或导入示例 JSON 数据。
4. 在“数据管理”中定期导出 JSON 备份。

### 导入示例数据

1. 打开应用。
2. 进入“数据管理”。
3. 点击“导入 JSON”。
4. 选择仓库中的 `demo.json` 文件。

## 数据说明

本项目采用 local-first（本地优先）设计，研究数据主要保存在当前浏览器的本地存储中。

请注意：

- 清除浏览器数据可能导致已保存的记录丢失。
- 更换浏览器、电脑或网页地址后，原数据不会自动同步。
- 请定期导出 JSON，并将备份保存在项目目录或云盘中。
- 当前为测试版本，不建议将唯一一份重要科研数据只保存在浏览器中。
